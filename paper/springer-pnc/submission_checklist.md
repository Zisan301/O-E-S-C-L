# Photonic Network Communications - Springer submission checklist

Prepared against the official Springer Nature submission guidelines for Photonic Network Communications.

## Journal requirements handled

- Springer LaTeX source prepared for `sn-jnl` with `twocolumn` option.
- Numeric citation style used.
- Abstract kept within 150-250 words.
- 6 keywords supplied.
- Figures named as `Fig1` to `Fig5` and cited in order.
- Tables cited in order and captioned.
- Source files and compiled preview PDF are included in the downloadable package.
- Statements and Declarations section added.
- Competing interests statement added.
- Funding statement added.
- Author contributions statement added.
- Data availability statement added.
- Code availability statement added with GitHub URL.
- Ethics approval and consent statements added.
- AI-assisted tools statement added for Springer transparency.
- Cover letter prepared.
- Supplementary CSV files prepared.

## Must edit before submission

- Replace placeholder author affiliations.
- Replace placeholder emails.
- Add ORCID IDs if available.
- Verify every DOI in `references.bib`.
- Download/use the official Springer LaTeX package from the journal page if local `sn-jnl.cls` is missing.
- Compile official `main.tex` with Springer class and upload both PDF and editable sources.
- Archive final code/data release in Zenodo or another repository for a persistent DOI if possible.

## Core limitation to keep in the manuscript

The C+S model uses a simplified lumped inter-band penalty. It is not full Raman-calibrated WDM/ISRS modeling and must not be described as experimental validation.
