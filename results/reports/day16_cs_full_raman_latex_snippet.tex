% Day-16 full Raman/ISRS-aware C+S validation snippet
The C+S scenario was evaluated using a combined C+S GNPy reference with Raman/ISRS-aware simulation settings. The calibrated held-out RMSE was 0.2549 dB and the maximum absolute held-out error was 0.3516 dB. Leave-one-power-out validation produced an RMSE of 0.2988 dB.
