\subsection{Numerical convergence and reference checking}
A Day-10 validation run was added for the accepted single-band operating points to reduce the risk that the PCS gain is an artifact of a short waveform simulation. The run repeated paired PCS-minus-uniform comparisons over larger symbol counts and compared the two largest symbol-count settings using an absolute and relative GMI-gain drift gate. The symbol-count stability gate passed for: C, S. In addition, the external reference file was not available. If no external GNPy/GN/EGN reference file is supplied, the analytical GSNR comparison is interpreted only as an internal sanity check and not as formal external validation.
